# Responsive Weather Dashboard

Platform Based Development laboratory activity built with React and Vite.

## Project structure

- `static.html` contains the static dashboard deliverable for Part 1.
- `src/` contains the dynamic React application.
- `docs/static-analysis.md` explains why the static version is static and its limitations.

## Local development

```bash
npm run dev
```

## Production build

```bash
npm run build
```
